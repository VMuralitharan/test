Add a button with the label "Click Me" and write JavaScript so that when clicked, an alert appears with "Button was clicked!"

<!-- HTML --> <button id="myButton">Click Me</button> <script> const button = document.getElementById("myButton"); button.addEventListener("click", function() { alert("Button was clicked!"); }); </script>

Answer 1(e)(1)

7 marks

Write a JavaScript function named addNumbers that takes two numbers as input and returns their sum. Call it with 5 and 10 and display the result.

<script> function addNumbers(num1, num2) { return num1 + num2; } let sum = addNumbers(5, 10); document.getElementById("result").textContent = "The sum is: " + sum; </script>
